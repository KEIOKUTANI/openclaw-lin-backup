## OpenClaw設定方法

### 方法1: モデル名指定（推奨）

メッセージで明示的に指定:
```
@haiku これ実行して
@sonnet この複雑なバグ直して
```

### 方法2: デフォルトモデル変更

```bash
# Haikuをデフォルトに
openclaw config set model anthropic/claude-haiku-3.5

# Sonnetに戻す
openclaw config set model anthropic/claude-sonnet-4
```

### 方法3: セッションごとに切り替え

```bash
# Haikuセッション開始
/new haiku

# Sonnetセッション開始  
/new sonnet
```
