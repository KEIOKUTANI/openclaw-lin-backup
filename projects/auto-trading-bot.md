# Auto Trading Bot Project

**開始日**: 2026-02-12  
**ステータス**: 構想・研究フェーズ

## プロジェクト概要

AIエージェント（OpenClaw）が市場情報を自動分析し、自動売買を実行するシステム。

## 現在の構想

### 1. プラットフォーム
- **未定** - TradingViewか他のプラットフォームか研究中
- 選定基準: API連携の容易さ、データの質、コスト

### 2. 取引対象
- **仮想通貨**: Worldcoin (WLD) など
- **予測市場**: Polymarket
- **その他**: 柔軟に拡張可能
- **複数銘柄対応**: 必須

### 3. 軍資金
- 東京ポイント: 11,000ポイント
- Worldcoin: 5,000分
- **合計**: 約 $1,850 USD相当（WLD @ $0.37換算）

### 4. リスク管理（バンクロール管理）
- ポーカーの概念を応用
- **標準**: 20倍ルール（資産の1/20 = 5%）
- **運用想定**: 50-60倍ルール（資産の1/50-1/60 = 1.67-2%）
- ケリー基準の考え方を採用
- 破産リスクを最小化

**1回あたりの取引額目安**:
- 20倍ルール: ~$92.5 per trade
- 50倍ルール: ~$37 per trade
- 60倍ルール: ~$30.8 per trade

### 5. 分析手法
- **テクニカル分析**: チャート、指標
- **ニュース分析**: 準備中
- **ファンダメンタル**: 必要に応じて

### 6. 実行フロー（想定）
```
情報収集 → AI分析 → 判断 → 自動実行 → 記録・評価
```

## 技術的選択肢（検討事項）

### プラットフォーム候補
1. **TradingView**
   - ✅ チャート分析に強い
   - ✅ Pine Script で戦略記述
   - ❌ 直接取引実行は不可（取引所APIと併用必要）

2. **取引所 API 直接**
   - Binance, Bybit, Coinbase, OKX など
   - ✅ 直接実行可能
   - ❌ チャート分析は自前実装

3. **Polymarket API**
   - 予測市場特化
   - ✅ ユニークな市場アクセス
   - ❌ 仮想通貨取引所とは別枠

4. **ハイブリッド**
   - TradingViewでチャート分析
   - 取引所APIで実行
   - ✅ 両方の強みを活用

### OpenClaw 統合アプローチ
1. **Browser automation** (browser tool)
   - TradingView画面をスクレイピング
   - チャートのスナップショット分析

2. **API integration**
   - 取引所APIを直接叩く
   - リアルタイムデータ取得

3. **Cron + Heartbeat**
   - 定期実行（5分足、15分足など）
   - 急変時のアラート

4. **Sub-agent spawning**
   - 重い分析は isolated session で実行
   - メインセッションは軽量に保つ
