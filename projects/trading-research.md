## 🎯 Polymarket詳細

### なぜPolymarketが面白いのか？

1. **ビットコイン4年サイクルから独立**
   - 仮想通貨が弱気でも、予測市場は常に機会がある
   - 政治、スポーツ、経済指標、ニッチイベント

2. **公式のAIエージェントサポート**
   - GitHub: `Polymarket/agents` - 公式Python SDK
   - Gamma API（市場データ）+ CLOB API（注文）
   - リアルタイムデータストリーム（WebSocket）

3. **Bot事例が豊富**
   - Automated Market Making（流動性提供で報酬）
   - Copy Trading（優秀トレーダーを自動フォロー）
   - 裁定取引bot

4. **Aaron (Coin Bureau) の戦略**
   - 「有益なウォレットを分析 → 何が機能しているか逆算 → 戦略をテスト」
   - 低ボリューム市場での優位性探し

### Polymarket API構造

- **Gamma API**: 市場メタデータ、価格、ボリューム、流動性
- **CLOB API**: 注文執行、オーダーブック
- **RTDS API**: リアルタイムデータストリーム
- **x402 統合**: AIエージェントがUSDCで直接支払い可能（Stripe対応）

### コスト
- **データ取得**: $0.01 USDC/リクエスト（CoinGecko × x402経由）
- **取引手数料**: マーケットによる（通常1-2%）

## 🛠️ 技術的選択肢

### Option A: TradingView + 取引所API (ハイブリッド)

**構成:**
```
TradingView (チャート分析)
    ↓
OpenClaw (判断)
    ↓
取引所API (実行: Binance / Bybit / Hyperliquid)
```

**メリット:**
- TradingViewの強力なチャートツール
- OpenClawがブラウザ自動化でTVを操作可能
- 取引所APIで直接執行

**デメリット:**
- 2つのシステムを統合する必要

### Option B: 取引所API直接 + データフィード

**構成:**
```
市場データAPI (CoinGecko / Binance Data)
    ↓
OpenClaw (分析 + 判断)
    ↓
取引所API (Binance / Bybit / Hyperliquid)
```

**メリット:**
- シンプル
- レイテンシーが低い
- コスト効率良い

**デメリット:**
- チャート分析は自前実装

### Option C: Polymarket専用

**構成:**
```
Polymarket Gamma API (市場データ)
    ↓
OpenClaw (ニュース分析 + 判断)
    ↓
Polymarket CLOB API (実行)
```

**メリット:**
- 仮想通貨市場と相関が低い
- ニッチな機会を狙える
- 公式SDK充実

**デメリット:**
- 流動性は仮想通貨取引所より低い
- 米国からのアクセス制限

### Option D: マルチプラットフォーム (Aaron方式)

**構成:**
```
OpenClaw Hub
   ├─ 仮想通貨取引所 (Hyperliquid / Binance)
   ├─ Polymarket (予測市場)
   └─ Solana DEX (アービトラージ)
```

**メリット:**
- リスク分散
- 複数市場での機会
- 市場相関が低い = ポートフォリオ効率良い

**デメリット:**
- 複雑
- 管理コスト高

## 📈 バンクロール管理の実装

### Kelly基準の応用

あなたの提案（50-60倍ルール）は合理的。

**計算式:**
```
f* = (bp - q) / b
```
- `f*` = 最適賭け金比率
- `b` = オッズ（ペイオフ倍率 - 1）
- `p` = 勝率
- `q` = 負け率 (1 - p)

**フル・ケリー vs フラクショナル・ケリー:**
- フル: リターン最大化だが、ボラティリティ高い
- 1/4 Kelly: より安全（破産確率激減）
- あなたの50-60倍 ≈ 約1/2.5 〜 1/3 Kelly（保守的でGood👍）

### 実装例

**軍資金**: $1,850
- **20倍ルール** (標準): $92.5 / trade
- **50倍ルール** (あなた): $37 / trade ✅
- **60倍ルール** (超保守): $30.8 / trade ✅

**ストップロス**: 
- 日次損失上限: -5% (軍資金の)
- 週次損失上限: -10%
- 月次損失上限: -20%

**利確ルール**:
- ターゲット1: +10% → 50%利確
- ターゲット2: +20% → 残り50%利確
- トレーリングストップ: ピークから-5%

## 🔐 セキュリティ・リスク管理

### OpenClaw「Poisoned Plugin」事件

- **2026年1月**: 悪意あるskillがユーザーの資金を盗む事件発生
- **教訓**: サードパーティのskillは慎重に検証
- **対策**:
  - 公式skill/信頼できる開発者のみ使用
  - APIキーは読み取り専用 or 出金制限付き
  - 定期的な監査

### API権限設定

**推奨設定:**
- ✅ 読み取り (Read)
- ✅ 取引 (Trade)
- ❌ 出金 (Withdraw) ← 絶対に無効化

**IP制限**: 可能ならOpenClaw稼働マシンのIPのみ許可
