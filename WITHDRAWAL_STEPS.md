# Polymarket 出金手順

## 目的
Polymarketの残高 $27.81 を、スクリプトが認識できるウォレットに移動

## 手順（5分）

### 1. Polymarketで出金画面を開く
- 画面右上の「デポジット」ボタンの隣にある「出金」をクリック
- または Portfolio → 出金

### 2. 送金先アドレスを入力
```
0x116545fd27202167f3993887505157d5E8Be6B65
```

**または「Use connected」ボタンを押す**
→ ただし、これは `0x1011cd...` が入力されるので、手動で上記アドレスに変更してください

### 3. 金額を入力
- Amount: **27.81** （または「Max」ボタンをクリック）
- Token: **USDC**
- Network: **Polygon**

### 4. 確認して実行
- 「Withdraw」ボタンをクリック
- MetaMaskで承認
- トランザクション完了を待つ（30秒〜2分）

### 5. 完了確認
Polygonscanで確認：
```
https://polygonscan.com/address/0x116545fd27202167f3993887505157d5E8Be6B65
```

---

## 完了したら
Telegramで「完了しました」とメッセージを送ってください。
すぐにテスト取引を実行します。

---

**注意事項：**
- ガス代（数円）がかかります
- Polymarket内部 → ウォレット本体への移動です
- 移動後、スクリプトから直接取引できるようになります
