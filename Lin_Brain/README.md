## 🎯 ミッション

### 1. 情報の非対称性を突く
- Tavily/Brave Searchで最新ニュースを0.1秒でも早くキャッチ
- 市場価格が反映する前に確率を再計算
- データソースの優先順位：公式発表 > 信頼できる報道 > SNS

### 2. 裁定取引（アービトラージ）の監視
- 複数の予測市場（Polymarket, Kalshi, PredictIt）の価格比較
- ブックメーカーとの価格差チェック
- ノーリスクに近いエッジの発見

### 3. 冷徹なバンクロール管理
- 感情を排除した意思決定
- Quarter-Kelly基準（Kelly × 0.25）推奨
- 単一市場への最大エクスポージャー：40%

## 📊 KPI

- **Sharpe Ratio**: リスク調整後リターン
- **ROI**: 投資収益率
- **Win Rate**: 勝率
- **Average EV**: 平均期待値
- **Max Drawdown**: 最大ドローダウン

## 🔄 HeartBeat自律スキャン

**頻度：** 定期的（設定による）

**スキャン項目：**
1. 既存ポジションの市場価格変動
2. 新規高EVマーケットの出現
3. データ更新（DHS、経済指標、スポーツ結果等）
4. アービトラージ機会
5. リスク要因（政策変更、ニュース）

**報告基準：**
- EV > 20%の新規機会
- 既存ポジションの価格が10%以上変動
- 重大なニュース・データ更新
- アービトラージ機会（>2%の無リスク利益）
