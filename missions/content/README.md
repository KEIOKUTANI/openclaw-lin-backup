# Content Lin - YouTubeコンテンツ制作ミッション

## 概要
ポーカーの「しくじり（Blunder）」解説動画等のYouTube台本を作成し、コンテンツ収益化を目指します。

## 目標
- ポーカー解説動画の台本作成
- しくじり（Blunder）分析コンテンツ
- 視聴者エンゲージメント最大化
- #YouTubeトピックで進捗管理

## コンテンツタイプ
1. **しくじり解説**: プロのミスプレイ分析
2. **GTO戦略**: 最適戦略の解説
3. **実践レビュー**: 実際のプレイ振り返り

## スクリプト
- `script_generator.py`: 台本生成
- `hand_analyzer.py`: ハンド分析
- `content_planner.py`: コンテンツカレンダー管理

## ワークフロー
1. ハンドヒストリー分析
2. Blunderの特定
3. 台本構成作成
4. セリフ・解説文生成
5. #YouTubeトピックへ報告

## データ保存先
- `data/scripts_*.md`: 台本
- `data/hand_history_*.json`: ハンドデータ
- `data/content_calendar.json`: 投稿計画

## 台本構成例
```markdown
# [タイトル] プロでも間違える！○○のBlunder解説

## イントロ (30秒)
- フック: 衝撃的なミスの紹介
- 今日の内容予告

## 状況説明 (1分)
- ハンド状況
- スタック・ポジション

## 分析 (3分)
- 実際のプレイ
- 正しいプレイ
- GTO的考察

## まとめ (30秒)
- 学びのポイント
- 次回予告
```
